# Retail Analytics Case Study — Snowflake + Streamlit + Tableau

## Project Includes

- Snowflake notebook for data loading and SQL analysis
- Streamlit app for interactive analytics
- Tableau public dashboard
- Retail Superstore dataset analysis

## Tech Stack

- Snowflake
- Snowpark Python
- Streamlit
- Pandas
- Matplotlib / Seaborn
- Altair
- Tableau Public

## Files in this Repo

- snowflake_analysis.ipynb → Snowflake notebook
- streamlit_app.py → Streamlit app
- requirements.txt → Python dependencies

## Tableau Dashboard

https://public.tableau.com/views/Retail-Superstore/Retail?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

## Run Streamlit App Locally

Install dependencies:

pip install -r requirements.txt

Run app:

streamlit run streamlit_app.py

## Purpose

The goal of this project is to analyze retail sales and profitability by region, category, and segment, and present insights using dashboards and interactive tools.
